# LUFFY AI - Project Roadmap

## Phase 1: Scaffolding (COMPLETED)
- Basic UI layout, Electron setup, FastAPI backend structure.

## Phase 2: Design Language (COMPLETED)
- Styling, theme, and component foundations based on Kinetic Obsidian design system.

## Phase 3: Backend Core (COMPLETED)
- Ollama provider integration, settings, logger, health checks.

## Phase 4: Chat (COMPLETED)
- Chat integration, token streaming, markdown rendering, Zustand state management.
- *Known Limitation:* Time To First Token (TTFT) is approximately 12–15 seconds during local Ollama generation. This is accepted for now. Optimization is deferred to Phase 11.

## Phase 5: Memory (COMPLETED)
- SQLite, Vector DB (ChromaDB), RAG implementation.

## Phase 6: Voice (PENDING)
- Realtime Audio API integration, Speech-to-Text, Text-to-Speech.

## Phase 7+ (PENDING)
- Tools, Settings, OS integration, and advanced features.
